Conceitos CSS

Perguntas:

  • A- É possível sobrepor à aparência(estilo)definida em regras CSS anteriores, mais genéricas, por regras mais específicas? Se possível, essa técnica tem nome?Exemplifique.

    • Sim, o nome da técnica é Overriding.

    • Exemplificação

      No código HTML eu tenho um marcador(H1), e quero que um dos atributos dele como por exemplo a cor, seja diferenciada de todo o body, então em um código CSS usando a técnica Overriding atribuindo a ele um fator mais específico, isso será possivel.

  • B- Caso definíssemos uma regra CSS mais geral para cabeçalhos de nível 1, 2 e 3, seguida de outra somente para o nível 1, como seria o comportamento do navegador? Descreva os vários cenários possíveis e o comportamento do navegador em cada uma delas.

    • Assim como relata a exemplificação da letra A, um style mais específico será aplicado pelo navegador ao invés de um style mais geral.

    • Alguns Dos Possíveis Cenários

      |1º| Caso especifique um tipo de fonte padrão(Arial) e outra para o [H1](Verdana), a fonte Verdana permanecerá no mesmo.

      |2º| Caso especifique uma cor padrão(Azul) e outra para o [H1](Preta), a cor Preta permanecerá no mesmo.

  • C- É sempre interessante otimizar o código CSS? Cite algumas vantagens.

    • É importante haver otimização do código.

    • Vantagens

      |1|Clareza no código.

      |2|Evitar duplicação no código.

      |3|Reduzir o tempo de produção do código.

      |4|Reduzir tempo de redenrização do navegador.

  • D- Defina herança em CSS. Utilize, também,imagens para ilustrar tais definições.

    • Ultilizando a herança, sendo definido um style, ou seja, uma característica em um marcador geral, todas as tags interiores herdam tal style. Porém não impede de ser alterado pelo método Overriding.

    • Exemplo

      Após um style ser atribuído ao Body, todos aqueles que estam abaixo herdam o mesmo style, desde suas tags necessitem.

      Código Demostrativo

      body {
      font-family: sans-serif;}


      h1, h2{
      color: gray;
      font-family:sans-serif;}
      //Não há mais necessidade pois foi herdado!

  • E- Podemos sobrescrever características herdadas todas as vezes que desejarmos deixá-la mais específica?

    • Os argumentos mais específicos definem sempre a parte final de algo, ou seja, o estilo final de tal tag. Logo podemos confirmar que sim podemos sobrescrever características herdadas.
